#  AI Course Generator & Learning Platform

This is a full-stack **Course Creation and Learning Platform** built with **Next.js 14 (App Router)**, **Drizzle ORM**, **PostgreSQL**, **Tailwind CSS**, and **Clerk Authentication**.

It allows admins or creators to **generate structured programming courses using AI (Gemini API)**, embed **YouTube videos** via the **YouTube Data API**, and manage dynamic course and chapter content with ease.
